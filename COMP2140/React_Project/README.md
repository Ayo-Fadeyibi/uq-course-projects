Use of Generative AI (ChatGPT) in Development

During the development of this project, Generative AI (ChatGPT) was used as a coding assistant to support:

Debugging: Fixing issues such as

state not resetting in forms,

missing dependency errors in useEffect,

conditional rendering problems (e.g., showing “There are no questions” when no questions exist).

Refactoring & best practices: Improving readability with comments, docstrings, and modular component design.

Styling & animation help: Suggestions for integrating Framer Motion animations, hover effects, and improved UI/UX consistency.

Documentation: Generating docstrings, component explanations.
